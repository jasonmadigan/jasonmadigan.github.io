Common Questions

Where are my files saved?
By default, Snag saves files to your Downloads folder. You can change the save location for each transfer or set a default in Preferences.
How do I resume a failed transfer?
Snag automatically retries failed chunks with configurable retry delays. If a transfer shows an error, right-click and select Resume, or it will retry on its own.
What are .snag files?
Temporary bundles that store partial transfer data and progress. They are automatically removed when a transfer completes. If you quit Snag mid-transfer, the .snag file preserves your progress for next time.
How do I use cookie authentication?
In the Add Transfer sheet, expand Authentication, select Cookies.txt, and paste your cookies in Netscape cookies.txt or JSON format (from browser extensions like EditThisCookie).
How do I change the number of threads?
Set the default in Preferences > Transfers, or configure per-transfer in the Add Transfer sheet. You can also change threads on active transfers via right-click > Edit Settings.
