### Common Questions

**Where are my files saved?**
By default, Snag saves files to your Downloads folder. You can change the save location for each download or set a default in Preferences.

**How do I resume a failed download?**
Snag automatically retries failed chunks with configurable retry delays. If a download shows an error, right-click and select Resume, or it will retry on its own.

**What are .snag files?**
These are temporary bundles that store partial download data and progress. They're automatically removed when a download completes. If you quit Snag mid-download, the .snag file preserves your progress for next time.

**How do I use cookie authentication?**
In the Add Download sheet, expand Authentication, select Cookies.txt, and paste your cookies in Netscape cookies.txt or JSON format (from browser extensions like EditThisCookie).

**How do I change the number of threads?**
Set the default in Preferences > Downloads, or configure per-download in the Add Download sheet under Download Settings. You can also change threads on active downloads via right-click > Edit Settings.
